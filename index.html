Linear Algebra

Scalars, Vectors, Matrices & Tensors

Scalar
A single number, e.g. x ∈ ℝ. Used for learning rate, loss value, weight.
Vector
An ordered array of numbers: v ∈ ℝⁿ. Represents features or embeddings.
Matrix
2D array: A ∈ ℝᵐˣⁿ. Represents weight layers in neural networks.
Tensor
N-dimensional generalization. Images are rank-3 tensors (H × W × C).

Vector Operations

Dot Product: a · b = Σᵢ aᵢbᵢ = |a||b| cos θ
Addition: c = a + b → cᵢ = aᵢ + bᵢ
Scalar Mult: λa → (λa)ᵢ = λaᵢ
L2 Norm: ‖x‖₂ = √(Σ xᵢ²)
L1 Norm: ‖x‖₁ = Σ |xᵢ|
Cross Product: a × b (3D only)
AI Use: Cosine similarity = (a·b)/(‖a‖‖b‖) — used in word embeddings, recommendation systems, and semantic search.

Matrix Operations

Matrix Mult: C = AB, Cᵢⱼ = Σₖ AᵢₖBₖⱼ
Transpose: (Aᵀ)ᵢⱼ = Aⱼᵢ
Inverse: AA⁻¹ = I (only for square, non-singular)
Hadamard: (A ⊙ B)ᵢⱼ = AᵢⱼBᵢⱼ (element-wise)
Trace: tr(A) = Σᵢ Aᵢᵢ
Determinant: det(A) — measures volume scaling
  • Forward pass in neural nets: z = Wx + b
  • Backprop uses transposed weight matrices
  • Attention: Attention(Q,K,V) = softmax(QKᵀ/√d) V

Eigenvalues & Eigenvectors

Av = λv
Characteristic equation: det(A − λI) = 0
Eigendecomposition: A = QΛQ⁻¹
Symmetric: A = QΛQᵀ (Q orthogonal)
PCA finds eigenvectors of the covariance matrix. Largest eigenvalue → direction of most variance. Used for dimensionality reduction before feeding to ML models.

Singular Value Decomposition (SVD)

A = UΣVᵀ
U ∈ ℝᵐˣᵐ (left singular vectors, orthogonal)
Σ ∈ ℝᵐˣⁿ (diagonal — singular values σ₁ ≥ σ₂ ≥ ... ≥ 0)
Vᵀ ∈ ℝⁿˣⁿ (right singular vectors, orthogonal)
Low-rank approx: Aₖ = Σᵢ₌₁ᵏ σᵢ uᵢ vᵢᵀ
Truncated SVD is used in recommender systems (matrix factorization), NLP (LSA), and data compression. It's the backbone of collaborative filtering.

Calculus & Derivatives

Limits & Continuity

lim(x→a) f(x) = L
f is continuous if: lim(x→a) f(x) = f(a)
Differentiability ⟹ Continuity (converse not always true)

Derivatives — Rules

Power Rule: d/dx[xⁿ] = nxⁿ⁻¹
Product Rule: d/dx[f·g] = f'g + fg'
Quotient Rule: d/dx[f/g] = (f'g − fg') / g²
Chain Rule: d/dx[f(g(x))] = f'(g(x)) · g'(x)
Exponential: d/dx[eˣ] = eˣ
Log: d/dx[ln x] = 1/x
Sigmoid: σ'(x) = σ(x)(1 − σ(x))
Tanh: tanh'(x) = 1 − tanh²(x)
ReLU: f'(x) = 1 if x > 0, else 0
Chain Rule IS Backpropagation. Every layer gradient is computed by chaining local derivatives from output to input: ∂L/∂w = ∂L/∂a · ∂a/∂z · ∂z/∂w

Activation Function Derivatives

ActivationFormulaDerivativeNotes

Tanh(eˣ−e⁻ˣ)/(eˣ+e⁻ˣ)1−tanh²(x)Zero-centered, still vanishes
ReLUmax(0, x)0 or 1Fast, dying neuron problem
Leaky ReLUmax(αx, x)α or 1Fixes dying ReLU
GELUx·Φ(x)Φ(x) + x·φ(x)Used in BERT, GPT

Taylor Series & Approximation

f(x) ≈ f(a) + f'(a)(x−a) + f''(a)(x−a)²/2! + ...
eˣ = 1 + x + x²/2! + x³/3! + ...
ln(1+x) ≈ x − x²/2 + x³/3 − ... for |x| < 1
Taylor expansion justifies gradient descent: first-order approximation of the loss landscape. Second-order methods (Newton) use the Hessian (second derivative).
03

Multivariable Calculus

Partial Derivatives & Gradient

Partial: ∂f/∂xᵢ — rate of change w.r.t. xᵢ (others held constant)
Gradient: ∇f(x) = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ
Directional Derivative: D_v f = ∇f · v̂
∇f points in direction of steepest ASCENT
In gradient descent: w ← w − η ∇_w L. We subtract because we want to minimize loss (steepest descent).

Jacobian Matrix

For f: ℝⁿ → ℝᵐ
J ∈ ℝᵐˣⁿ, Jᵢⱼ = ∂fᵢ/∂xⱼ
Backprop = multiplying Jacobians across layers
δL/δx = Jᵀ · δL/δf

Hessian Matrix

H ∈ ℝⁿˣⁿ, Hᵢⱼ = ∂²f / (∂xᵢ ∂xⱼ)
Symmetric if f is twice-differentiable
If H is positive definite → local minimum
If H is negative definite → local maximum
If H has mixed signs → saddle point
Saddle points are common in deep learning. SGD with momentum escapes them. The Hessian eigenspectrum explains why some directions learn faster than others.

Chain Rule (Vector Form)

∂L/∂W = ∂L/∂z · ∂z/∂W (backpropagation in matrix form)
For z = Wx + b:
∂L/∂W = δᵀxᵀ (outer product)
∂L/∂x = Wᵀδ
∂L/∂b = δ

Integration — Key Results for AI

∫ p(x) dx = 1 (probability normalization)
E[X] = ∫ x p(x) dx
Var[X] = ∫ (x − μ)² p(x) dx
Gaussian: ∫₋∞^∞ e^(−x²) dx = √π

Probability Theory

Fundamentals

Sample Space Ω
All possible outcomes. E.g., Ω = {H, T} for a coin.
Event A
A subset of Ω. P(A) ∈ [0,1], P(Ω) = 1.
Complement: P(Aᶜ) = 1 − P(A)
Union: P(A∪B) = P(A) + P(B) − P(A∩B)
Intersection: P(A∩B) = P(A|B)·P(B) = P(B|A)·P(A)
Independence: P(A∩B) = P(A)·P(B)

Conditional Probability & Bayes' Theorem

P(A|B) = P(A∩B) / P(B)

Bayes' Theorem:
P(A|B) = P(B|A)·P(A) / P(B)

Full form (Law of Total Probability):
P(B) = Σₖ P(B|Aₖ)·P(Aₖ)
Posterior = (Likelihood × Prior) / Evidence
Core of Bayesian ML, Naive Bayes classifier, and probabilistic graphical models. GPT predicts P(next token | context) — that's conditional probability.

Expectation & Variance

E[X] = Σ x·P(X=x) or ∫ x·f(x)dx
E[aX + b] = aE[X] + b
E[X + Y] = E[X] + E[Y] (linearity, always holds)
E[XY] = E[X]E[Y] (only if independent)

Var[X] = E[X²] − (E[X])²
Var[aX + b] = a²Var[X]
Std Dev: σ = √Var[X]

Covariance: Cov(X,Y) = E[(X−μₓ)(Y−μᵧ)]
Correlation: ρ = Cov(X,Y) / (σₓσᵧ) ∈ [−1, 1]

Important Theorems

Law of Large Numbers
As n→∞, sample mean x̄ → true mean μ. Foundation of Monte Carlo methods.
Central Limit Theorem
Sum of many independent RVs → Normal distribution, regardless of individual distributions. Justifies Gaussian noise assumptions.
Markov's Inequality
P(X ≥ a) ≤ E[X]/a for X ≥ 0. Used in generalization bounds.
Chebyshev's Inequality
P(|X − μ| ≥ kσ) ≤ 1/k². Bounds outlier probability.
05

Statistics

Descriptive Statistics

Mean: μ = (1/n) Σ xᵢ
Median: Middle value when sorted
Mode: Most frequent value
Variance: s² = (1/(n−1)) Σ (xᵢ − x̄)² (sample)
Skewness: E[(X−μ)³/σ³] — asymmetry
Kurtosis: E[(X−μ)⁴/σ⁴] − 3 — tail heaviness

Maximum Likelihood Estimation (MLE)

θ_MLE = argmax_θ P(D|θ) = argmax_θ Σ log P(xᵢ|θ)

For Gaussian: μ_MLE = (1/n)Σxᵢ, σ²_MLE = (1/n)Σ(xᵢ−μ)²
Log-likelihood: ℓ(θ) = Σᵢ log p(xᵢ; θ)
Training a neural network by minimizing cross-entropy loss IS doing MLE. The loss function is the negative log-likelihood.

Maximum A Posteriori (MAP)

θ_MAP = argmax_θ P(θ|D) = argmax_θ [log P(D|θ) + log P(θ)]
Gaussian prior → L2 regularization (Ridge)
Laplace prior → L1 regularization (Lasso)
Regularization = MAP estimation with a prior on weights. L2 penalty ‖w‖² comes from a Gaussian prior N(0, σ²).

Bias-Variance Tradeoff

Expected Error = Bias² + Variance + Irreducible Noise

Bias: Error from wrong model assumptions (underfitting)
Variance: Sensitivity to training data fluctuations (overfitting)
Noise: Irreducible error in data
ModelBiasVarianceSolution
Too simpleHighLowMore capacity, features
Too complexLowHighRegularization, more data
OptimalLowLowBalanced

Hypothesis Testing & Confidence Intervals

p-value: Probability of observing data as extreme as ours under H₀
p < 0.05 → reject null hypothesis

95% CI: x̄ ± 1.96 · (σ/√n)
t-statistic: t = (x̄ − μ₀) / (s/√n)

Optimization

Gradient Descent Variants

Gradient Descent (GD): w ← w − η ∇_w L(w; D)
Stochastic (SGD): w ← w − η ∇_w L(w; xᵢ,yᵢ)
Mini-batch: w ← w − η ∇_w L(w; B) |B|≪n

SGD with Momentum: v ← βv + ∇L; w ← w − ηv
Nesterov: v ← βv + ∇L(w−βv); w ← w − ηv
Momentum accumulates velocity in consistent gradient directions, damping oscillations. β ≈ 0.9 is a typical value. Helps escape local minima and saddle points.

Adaptive Optimizers

Adam Full:

m̂ₜ = mₜ/(1−β₁ᵗ) (bias correction)
v̂ₜ = vₜ/(1−β₂ᵗ) (bias correction)
wₜ = wₜ₋₁ − η · m̂ₜ/(√v̂ₜ + ε)

Learning Rate Schedules

Step decay: η = η₀ · γ^(epoch/step_size)
Exponential: η = η₀ · e^(−λt)
Cosine anneal: η = η_min + 0.5(η_max − η_min)(1 + cos(πt/T))
Warmup: Linear ramp from 0 → η₀ then decay
Cyclical LR: Triangular oscillation between [η_min, η_max]

Convexity & Optimality Conditions

Convex function: f(λx + (1−λ)y) ≤ λf(x) + (1−λ)f(y)
First-order condition (minimum): ∇f(w*) = 0
Second-order (local min): H(w*) is positive semi-definite

Lagrangian (constrained): L(w,λ) = f(w) + Σλᵢgᵢ(w)
KKT conditions: ∇f + Σλᵢ∇gᵢ = 0, λᵢ ≥ 0, λᵢgᵢ = 0
Deep learning loss surfaces are non-convex. However, over-parameterized networks often have no "bad" local minima — most local minima are near global minima in practice.

Regularization Techniques

L2 (Ridge): L_reg = L + λ‖w‖₂² → Gaussian prior
L1 (Lasso): L_reg = L + λ‖w‖₁ → Laplace prior → sparsity
Elastic Net: L_reg = L + λ₁‖w‖₁ + λ₂‖w‖₂²
Dropout: Randomly zero p fraction of neurons during training
Batch Norm: ẑ = (z − μ_B)/σ_B · γ + β
Early Stop: Halt when val loss stops improving

Information Theory

Entropy

Shannon Entropy: H(X) = −Σ p(x) log₂ p(x) [bits]
Nat units: H(X) = −Σ p(x) ln p(x) [nats]
Maximum entropy: Uniform distribution → H = log(n)
Minimum entropy: Deterministic → H = 0
Entropy measures uncertainty. High entropy = more information in knowing the outcome. Used in decision trees (information gain = entropy reduction).

Cross-Entropy & KL Divergence

Cross-Entropy: H(p,q) = −Σ p(x) log q(x)
KL Divergence: KL(p‖q) = Σ p(x) log(p(x)/q(x)) ≥ 0
Relation: H(p,q) = H(p) + KL(p‖q)

KL(p‖q) ≠ KL(q‖p) (asymmetric!)
KL = 0 iff p = q
Cross-entropy loss for classification: L = −Σᵢ yᵢ log(ŷᵢ). Minimizing cross-entropy ≡ minimizing KL divergence between true and predicted distributions. It's the backbone of all classifier training.

Mutual Information

I(X;Y) = KL(p(x,y) ‖ p(x)p(y))
= H(X) − H(X|Y) = H(Y) − H(Y|X)
= H(X) + H(Y) − H(X,Y)
I(X;Y) ≥ 0, I(X;Y) = 0 iff X,Y independent
Mutual information measures how much X tells us about Y. Used in feature selection, contrastive learning (maximizing MI between views), and variational autoencoders.

Numerical Methods & Tricks

Numerical Stability

Softmax (numerically stable):
softmax(x)ᵢ = e^(xᵢ − max(x)) / Σⱼ e^(xⱼ − max(x))

Log-Sum-Exp:
log Σ eˣⁱ = c + log Σ e^(xᵢ−c), c = max(x)

Avoid: log(0), division by 0 → add ε = 1e-8
